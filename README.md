# 🧠 React Quiz App

A modern and interactive **Quiz Application** built with **React**.
This project allows users to answer quiz questions, track their score, and view results in real-time.

The app uses modern React concepts such as **Hooks, Context API, Reducer Hook**, and integrates **Firebase** for backend services.

---

## 🚀 Features

* 📚 Multiple Quiz Questions
* ⚡ Real-time score tracking
* 🎯 Result page with total score
* 🔄 Dynamic data fetching using Fetch API
* 🔐 Firebase integration
* 🎨 Responsive UI with Tailwind CSS
* 🧩 State management using Context API + useReducer
* ♻️ Reusable custom hooks
* 💅 Custom CSS styling

## ⚙️ Installation

Clone the repository

```
git clone https://github.com/yourusername/react-quiz-app.git
```

Go to project folder

```
cd react-quiz-app
```

Install dependencies

```
npm install
```

Run the project

```
npm run dev
```
